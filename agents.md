# Agents Guidelines

- Use TypeScript strict mode for all code.
- Provide types where they cannot be inferred
- do not rely on cdn's
- all features must work with github pages
- always protect agains XSS attacks

# Test Guidelines
- always use test driven development.
- always update the tests when making changes.

# style guidelines
- Use tailwind primitives for all styles
- use a greyscale pallette throughout
- use the cousine font
- support three form factors: phone, tablet, monitor
